End-of-session update ritual. Complete ALL of the following steps:

1. Update docs/PROGRESS.md:
   - Add a new session entry with today's date
   - List what was done this session
   - List any decisions made (and verify they're in docs/DECISIONS.md)
   - List current blockers
   - List what to build next

2. Update docs/MODULES.md:
   - Update the Status column for any module that changed
   - Update Known Issues column if any new issues were found

3. Update any docs/ files whose domain changed during this session:
   - If Immoscout selectors were discovered → update docs/BROWSER_AUTOMATION.md
   - If new bugs found → update docs/KNOWN_ISSUES.md
   - If data model changed → update docs/DATA_MODEL.md
   - If new decision made → append to docs/DECISIONS.md
   - If integration details changed → update docs/INTEGRATIONS.md

4. Update CLAUDE.md:
   - Update the "Current status" line with the latest module statuses
   - Update the "Last updated" timestamp

5. Confirm each file was saved by listing the files that were updated.
